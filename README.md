# Foodie Website  
A food delivery webpage.

## Features  
- Header with brand name and tagline  
- Hero section with promotional banner  
- Online restaurants listing  
- Restaurant cards with images and ratings  
- Food ranking list  
- Food culture information section  
- Pricing table for popular foods  
- Footer section  

## Technologies  
- HTML5  
- CSS3 (Flexbox, Box Model, Typography)
